## What Is DaisyDisk?

DaisyDisk is a disk space analyzer developed exclusively for macOS. The application scans your storage drives and presents the results using colorful, interactive visual maps that illustrate how files and folders occupy disk space.

Instead of digging through Finder manually or relying solely on macOS storage recommendations, DaisyDisk allows users to quickly identify what is consuming storage and decide which files should be removed.

The software supports:

* Internal Mac drives
* External hard drives
* SSDs
* USB storage devices
* Network volumes
* Cloud-mounted storage locations

Its primary goal is to provide transparency and control over storage management.

## How DaisyDisk Works

One of DaisyDisk's greatest strengths is its simplicity.

Users select the drive they want to analyze. DaisyDisk performs a scan that maps every folder and file occupying space.

### Step 2: Visualize Storage Usage

The software displays results using its signature sunburst diagram. Larger segments represent files or folders consuming more storage.

### Step 3: Explore File Structures

Clicking on sections of the visual map reveals deeper folder levels, allowing users to investigate storage usage interactively.

### Step 4: Identify Unnecessary Files

Users can locate:

* Large downloads
* Old backups
* Duplicate media
* Forgotten archives
* Unused application data
* Temporary files

### Step 5: Remove Selected Files

Files marked for deletion move to a collection area before final removal, providing an additional layer of control and safety.
